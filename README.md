### 字符编码转换
对于单个字符的编码，Python提供了ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符
eg: ord('a')    >>>97
    chr(97)     >>>'a'

###注意

* 在python缩进方面，按照约定俗成的管理，应该始终坚持使用4个空格的缩进
* 在python中是区分大小写的，如果写错了大小写那么程序就会报错
